# 3D Cosmos Simulation (sim-3d-cosmos)

An interactive 3D celestial observatory featuring:
- Real-time planet orbits and star constellations.
- Dynamic audio atmosphere using Web Audio API.
- Immersive UI with Cyber-aesthetic.
- Integrated visitor counter (Vercount.one).
- Optimized for Luna AI Hub with iframe scroll protocol.

## Tech Stack
- React 19
- Three.js
- Vite
- Tailwind CSS 4
- Lucide React
- Framer Motion

## Deployment
Hosted on GitHub Pages at [https://toydogcat.github.io/sim-3d-cosmos/](https://toydogcat.github.io/sim-3d-cosmos/) via GitHub Actions.
